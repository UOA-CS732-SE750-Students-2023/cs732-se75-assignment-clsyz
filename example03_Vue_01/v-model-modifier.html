The modifier for v-modle:

.number

Automatically converts user input to a number type, so we can calculate and add or subtract directly

.trim

Automatically filters whitespace characters from the beginning and the end of the user's input. It is possible that the user may have entered a space before the input. For this reason, we should remove the first whitespace characters.

.lazy

Update on "change" instead of "input", in other words, the user doesn't change over and over again as they type, but after the user has finished typing, when the input has lost its intersection.
